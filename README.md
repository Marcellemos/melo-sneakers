# Melo Sneakers

Pequeno projeto front-end: vitrine de tênis com HTML, CSS e JavaScript.

**Descrição**
- Projeto de exemplo com layout estático mostrando produtos (imagens em `img/`).

**Como usar**
- Abra `index.html` em um navegador moderno.
- Ou rode um servidor local (recomendado para evitar problemas com rotas/recursos):



**Estrutura do projeto**
- `index.html` — página principal
- `style.css` — estilos
- `script.js` — lógica/integrações
- `img/` — imagens dos produtos

- Faça um fork e envie um pull request com melhorias.
